## 3. Danh sách 10 phím tắt hữu ích
### Google Chrome:
1. Ctrl + Tab: Chuyển tab bên phải.
2. Ctrl + Shift + Tab: Chuyển tab bên trái.
3. Ctrl + F: Tìm kiếm nhanh trên trang.
4. Ctrl + D: Lưu trang vào dấu trang.
5. Alt + Home: Mở trang chủ trong tab hiện tại.

### Windows:
1. Win + E: Mở nhanh File Explorer.
2. Win + Mũi tên: Chia nhanh màn hình làm việc.
3. Win + I: Mở bảng Cài đặt hệ thống.
4. Win + Số (1,2,3...): Mở ứng dụng trên Taskbar.
5. Ctrl + Alt + Del: Mở các tùy chọn an toàn hệ thống.
